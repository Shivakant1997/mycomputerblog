📚 ओ लेवल (O Level) डिजिटल लाइब्रेरी

मार्गदर्शक: शिवाकांत चौधरी (प्रयागराज)

📘 मॉड्यूल 1: M1-R5 - IT Tools & Network Basics

परीक्षा के लिए सबसे महत्वपूर्ण अध्यायों का पूरा निचोड़:

1. कंप्यूटर का परिचय और पीढ़ियां
कंप्यूटर एक इलेक्ट्रॉनिक डिवाइस है। इसके मुख्य भाग CPU, ALU और मेमोरी यूनिट होते हैं। प्रथम पीढ़ी में वैक्यूम ट्यूब का उपयोग हुआ।
2. ऑपरेटिंग सिस्टम (OS)
यह यूज़र और हार्डवेयर के बीच माध्यम है। जैसे: Windows और Linux (ओपन-सोर्स)।
3. लिब्रेऑफिस राइटर
डॉक्यूमेंट टूल है। डिफ़ॉल्ट फाइल एक्सटेंशन .odt और बोल्ड शॉर्टकट Ctrl + B है।
4. लिब्रेऑफिस कैल्क
डेटा गणना टूल है। फाइल एक्सटेंशन .ods है और फार्मूला = से शुरू होता है।
5. लिब्रेऑफिस इम्प्रेस
स्लाइड प्रेज़ेंटेशन टूल है। फाइल एक्सटेंशन .odp और नई स्लाइड के लिए Ctrl + M है।
6. इंटरनेट और नेटवर्क
कंप्यूटर को नेटवर्क से जोड़ने के लिए IP Address और वेबसाइट खोलने के लिए Web Browser चाहिए।
7. डिजिटल वित्तीय उपकरण
ऑनलाइन लेनदेन के लिए UPI और सुरक्षा के लिए OTP जरूरी है।
📗 मॉड्यूल 2: M2-R5 - Web Designing & Publishing

वेब डिजाइनिंग के सभी महत्वपूर्ण अध्यायों का पूरा निचोड़:

1. HTML और टैग्स का परिचय
HTML से वेब पेज का स्ट्रक्चर बनता है। इसमें <ol> से नम्बर्ड और <ul> से बुलेट लिस्ट बनती है।
2. सीएसएस का परिचय (CSS)
इसका पूरा नाम Cascading Style Sheets है। यह पेज को सुंदर और रंग-बिरंगा बनाता है।
3. सीएसएस फ्रेमवर्क (W3.CSS)
W3.CSS एक मुफ़्त और रेस्पॉन्सिव फ्रेमवर्क है जो वेबसाइट को मोबाइल स्क्रीन के फिट कर देता है।
4. जावास्क्रिप्ट (JavaScript)
यह Client-Side Scripting भाषा है जो वेब पेज को डायनामिक और वर्किंग बनाती है।
5. फोटो एडिटर और वेब पब्लिशिंग
वेबसाइट को लाइव करने को Web Publishing कहते हैं, जिसके लिए FTP का उपयोग होता है।
📙 मॉड्यूल 3: M3-R5 - Programming & Problem Solving through Python

पाइथन प्रोग्रामिंग के सभी महत्वपूर्ण अध्यायों का पूरा निचोड़:

1. पाइथन का परिचय
Python एक सरल, हाई-लेवल और केस-सेंसिटिव भाषा है जिसका आविष्कार Guido van Rossum ने 1991 में किया था।
2. वेरिएबल्स और डेटा टाइप्स
डेटा स्टोर करने के लिए Variables का उपयोग होता है। मुख्य डेटा टाइप्स int (पूर्णांक), float (दशमलव), और str (टेक्स्ट) हैं।
3. डिसीजन मेकिंग (If-Else)
शर्तों के आधार पर निर्णय लेने के लिए if, elif, और else स्टेटमेंट का उपयोग किया जाता है।
4. लूप्स का उपयोग (Loops)
किसी कोड को बार-बार दोहराने के लिए For Loop और While Loop का उपयोग करते हैं।
for i in range(1, 6):
print(i)
5. लिस्ट और टुपल (List & Tuple)
• List: यह परिवर्तनीय (Mutable) होती है, इसे चौकोर ब्रैकेट [] से दर्शाते हैं।
• Tuple: यहअपरिवर्तनीय (Immutable) होती है, इसे छोटे ब्रैकेट () से दर्शाते हैं।
6. फंक्शन्स (Functions)
कोड को दोबारा इस्तेमाल करने के लिए ब्लॉक बनाया जाता है। पाइथन में फंक्शन बनाने के लिए def कीवर्ड का उपयोग होता है।
